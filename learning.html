Methods
refresh Void Refresh position of the element
options
parentObjectjQuery Objectcenter by parent object
positionTypestring(*)set css position: absolute, static, fixed, relative
directionAlignstring(*)center using set direction: vertical or v, horizontal or h, both
refreshWindowResizeboolean
delta : { left : number , top : number }move delta position from center
referenceObject : { top : jQuery Object , left : jQuery Object, bottom: jQuery Object, right: jQuery Object }use another object, for reference align.
(*) not necessary case sensitive.
